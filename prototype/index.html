inducta

rules

economy key

  • R = root
  • C0→C1→C2 = charge builds each turn
  • F→f→gone = flare decays if not used before your next turn

economy flow

  • empty → R advances turn
  • R → F or R → C0 has no turn cost
  • C2/C1/F/f → target advances turn
  • C0 is not usable yet

power

  • strong: C2, C1, F
  • normal: f
  • C2 used → C1
  • C1, F, f used → gone

combatant key

  • A→A* = arbiter arms, then fires
  • L2→L1→L0 = latcher counts down; L0 removes it
  • E→e→gone = executor breaches, then removes
  • + / ++ = sharper version of the same ask

combatant targets

  • L2/L1: power counts down toward L0
  • E: normal → e; strong → gone
  • e: normal or strong → gone
  • A*: resolves only while active; A is bad timing

combatant pressure

  • A* + arrow attacks the first eco hit in that direction
  • L* + arrow attacks the furthest eco in that direction
  • E/e + arrow moves forward, turns right when blocked, loads on eco, then crushes next shift

collapse

Full board + incoming pressure = collapse. Study state, not auto-reset.
